# MCP Tool (mcp__google_docs__google_docs)

Use this backend only when both the `meta google.docs` CLI and the Python script are unavailable.

The MCP tool `mcp__google_docs__google_docs` is self-describing — call it with no arguments or inspect its schema to see available actions and parameters. Its capabilities mirror the Python script but with less flexibility for batch operations and round-trip editing.

Do not prefer the MCP tool over the CLI or script just because it advertises itself as available.
